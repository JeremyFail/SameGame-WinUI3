Place MP3 files in this folder using the following names:

Sound effects (can be committed to the repo):
- remove1.mp3, remove2.mp3, remove3.mp3, etc. - tile removal variations (one is chosen at random)
- game-end.mp3 - game over

Optional background music (not committed; add before compiling):
- background1.mp3, background2.mp3, background3.mp3, etc. - long-form tracks (one is chosen at
  random; when it finishes, another random track starts; see .gitignore)

If a sound effect file is missing, the game falls back to built-in synthesized tones.
Background music controls appear in Advanced Options only when at least one background track is present.
